# 선택정렬
시간 복잡도O(n^2)

배열을 순회하면서 최솟값을 찾고 그 값을 앞에 값과 바꿈

이미 정렬된 첫번째 수를 제외하고 순회하면서 최솟값을 찾고 두번째 값과 바꿈

이를 배열 전체가 정렬될 때까지 반복
