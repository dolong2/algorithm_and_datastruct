# 선택 정렬

배열을 앞 부분부터  차례대로 이미 정렬된 부분과 비교하여 자신이 들어갈 위치에 삽입하는 알고리즘 

배열이 길어질수록 효율이 안 좋아짐

간단히 구현할 수 있음

버블 정렬과 선택 정렬에 비해서 빠르다

O(n^2)의 시간 복잡도를 가진다
